
# ADR Analytics Dashboard

## 📌 Overview

This project is an interactive Pharmacovigilance dashboard built using Microsoft Excel and Google Looker Studio.

It helps visualize Adverse Drug Reaction (ADR) reports through interactive charts and filters.

---

## 📊 Features

- Interactive Filters
- ADR Severity Analysis
- Drug-wise ADR Analysis
- Outcome Distribution
- Age-wise Analysis
- Time Trend Visualization
- Interactive Report Table

---

## 🛠 Tools Used

- Microsoft Excel
- Google Looker Studio

---

## 📁 Dataset

Simulated ADR dataset containing 60 reports.
